## frisbyjs
Use to test Rest API

## Install
1. npm install
2. npm install --save-dev frisby
3. npm install -g jasmine-node
4. cd your/project
5. jasmine-node test/

## Dependencies
[frisbyjs](http://frisbyjs.com)__
[jasmine-node](https://github.com/mhevery/jasmine-node)
